Tags HTML Basicas:

As tags mais importantes na HTML sao as tags que definem titulos, paragrafos e quebras de linha.

TITULOS:

Titulos sao definidos com as tags < h1 > a < h6 >. A < h1 > difine o titulo maior. A < h6 > difine o titulo menor.

Exemplos

< h1 >Este e um titulo< /h1 >

Este e um titulo

< h6 >Este e um titulo< /h6 >

Este e um titulo

PARAGRAFOS:

Os paragrafos sao definidos com a tag < p >

Exemplos


< p > Este e um paragrafo < /p>

Este e um paragrafo

QUEBRAS DE LINHAS:

A tag < br > e usada quando voce quer terminar uma linha, mas nao quer comecar um novo paragrafo.
A tag < br > forca a quebra de linha onde quer que voce a coloque.

Exemplos

< p > Este < br > e um para < br > grafo com quebras de linhas< /p >

Este
e um paragrafo
com quebras de linhas
